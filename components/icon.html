# Icon 图标

# 基础用法

通过name来设置不同的图标

<ivy-icon name="download"></ivy-icon>
<ivy-icon name="edit"></ivy-icon>
<ivy-icon name="reading"></ivy-icon>

# 旋转

添加spin属性来使 icon 旋转

<ivy-icon name="loading" spin></ivy-icon>
<ivy-icon name="refresh-right" spin></ivy-icon>
