# Select 下拉选择器

## 常规用法

<ivy-select>
    <ivy-option value="1" label="苹果"></ivy-option>
    <ivy-option value="2" label="香蕉"></ivy-option>
    <ivy-option value="3" label="橘子"></ivy-option>
    <ivy-option value="4" label="芒果"></ivy-option>
</ivy-select>

```html
<ivy-select>
    <ivy-option value="1" label="苹果"></ivy-option>
    <ivy-option value="2" label="香蕉"></ivy-option>
    <ivy-option value="3" label="橘子"></ivy-option>
    <ivy-option value="4" label="芒果"></ivy-option>
</ivy-select>
```

## 禁用选项

通过给 `select` 设置 `disabled` 属性来禁用下拉框

<ivy-select disabled>
    <ivy-option value="1" label="苹果"></ivy-option>
    <ivy-option value="2" label="香蕉"></ivy-option>
    <ivy-option value="3" label="橘子"></ivy-option>
    <ivy-option value="4" label="芒果"></ivy-option>
</ivy-select>
